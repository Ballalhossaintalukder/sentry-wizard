# Options
```
Options:
  --help     Show help                                                 [boolean]
  --version  Show version number                                       [boolean]
  --type     Choose a project type   [choices: "reactNative", "browser", "node"]
  --url, -u  The url to your Sentry installation [default: "https://sentry.io/"]
```
